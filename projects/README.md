# projects/

Your actual work lives here. Create these folders as you get to them:

- `cs50p/` — your CS50P problem-set solutions (Track 01).
- `leetcode/` — your problem-solving solutions, Python + Java (Track 03).
- …and anything else you build. **Build a lot. Build in public.**

**The rule:** every project is a Git project from day one. Commit early, commit often — even messy, even unfinished.
