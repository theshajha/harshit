# 🌐 Track 02 — Web Fundamentals (HTML · CSS · JS)

A **self-contained, curated web reference** — the building blocks of every website, in plain words you can scan any time. The goal of this track is the most satisfying one: **build things people can see**, starting with your own portfolio.

Two layers, like the Python track:
- **The notes (`notes/`) = the *understanding*.** Concept · mental model · cheat-sheet · example · gotchas.
- **Building things = the *doing*.** You learn web by making pages, not by reading about them. The notes tell you what to build.

---

## How to use this track
For each note:
1. **Read it** (5–10 min) — get the concept + cheat-sheet.
2. **Build the tiny thing it suggests** — open a `.html` file in your browser and try it.
3. **Fill the `✍️ Your turn` section** in your words.
4. **Commit** it.

Then pour it all into the **portfolio project** (note 07) — your real deliverable.
