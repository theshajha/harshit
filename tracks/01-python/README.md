# 🐍 Track 01 — Python Foundations

This track is a **self-contained, curated Python reference** — the things worth knowing, in plain words, that you can scan through any time. It's built so you don't *have* to go read ten external sites to understand something: the 80% lives right here.

Two layers, on purpose:

- **The notes (`notes/`) = the *understanding*.** Curated, scannable: concept · mental model · cheat-sheet · example · gotchas. This is your reference for years.
- **CS50P + exercises = the *doing*.** You can't learn to code by reading. The notes tell you what to go *do*; the doing is where the skill forms.

> 📌 The spine course for the *doing* is **CS50P** → <https://cs50.harvard.edu/python> (free to audit). The notes here roughly follow its order.

---

## How to use this track

For each topic:
1. **Read the note** here (5–10 min) — get the concept + the cheat-sheet.
2. **Do the matching CS50P lecture + problem set** — this is where it sticks.
3. **Fill the note's `✍️ Your turn` section** — in your own words, write the trickiest thing you learned + one example *you* wrote. Explaining it simply is how you actually own it.
4. **Commit** your filled note and your pset code (`projects/cs50p/`).

That loop — read → do → write it back in your words → commit — is the whole method.
