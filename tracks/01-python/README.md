# 🐍 Track 01 — Python Foundations (CS50P)

**The spine.** One course, start to finish: **CS50's Introduction to Programming with Python (CS50P)** — free, rigorous, problem-set driven. It builds the exact problem-solving muscle that everything later (backend, AI/ML, interviews) rewards.

🔗 **Course:** <https://cs50.harvard.edu/python> *(free to audit — you don't need to pay)*

## Why CS50P (and not a gentler course)
You've finished first-year CS, so you don't need "what's a variable." CS50P keeps you challenged, makes you **write and debug real code** every week, and pairs perfectly with the problem-solving track. It's the foundation that makes Python, backend, *and* AI/ML all easier later.

## How to work it
- Take **one "week" of CS50P at a time.** Watch the lecture, then **do the problem set** — the psets are where the actual learning happens.
- Keep your solutions in this repo under `projects/cs50p/`, and **commit each one**.
- **AI rule:** use it as a tutor (explain a concept, give a hint, review your code *after* you write it) — never as the thing that writes your psets. The handbook's "Habits" section explains why.
