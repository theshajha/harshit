# 🤖 Track 05 — AI / ML 🔜 coming next

This is where you're headed after the foundations. We'll build it out properly when you're ready — here's the arc so you can see the destination.

**The builder's approach:** you lean toward *building* over heavy math / research, so this track is weighted toward **concepts + practical building**, with *just enough* math to understand what's happening under the hood — not a theory-first slog.

## The arc (after Python + backend)
1. **Data stack** — NumPy, pandas, matplotlib. The daily tools of anyone doing ML.
2. **Just-enough math** — 3Blue1Brown's *Essence of Linear Algebra* + Khan Academy stats / probability. Visual, not a math degree.
3. **ML concepts** — **Andrew Ng's Machine Learning Specialization** (Coursera). *The* canonical "AI/ML concepts" course — Python-based and beginner-friendly.
4. **Practical / deep learning** — **fast.ai, "Practical Deep Learning for Coders"** — build-first, perfect for you.
5. **AI engineering** — building *with* models: LLMs, RAG, embeddings, vector databases, agents. (This is the "AI-systems literacy" pillar from the handbook — and where your backend skills pay off.)

## Not yet
