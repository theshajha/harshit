# This Week — Your First 7 Days

The goal this week isn't to master anything. It's to **get the machine running**: tools set up, code committed, and a first taste of the two muscles you'll build all year — **building** (web) and **problem-solving** (algorithms).

Quality over quantity. Finishing a few of these properly beats half-doing all of them.

---

## 0 · Setup — do this first
- [ ] Create a **GitHub account** (real name + photo) — it's your storefront for the next decade.
- [ ] Claim the **GitHub Student Developer Pack** with your `@learner.manipal.edu` email → <https://education.github.com/pack>  *(unlocks a pile of free tools — handbook Section 05 has the full list)*
- [ ] Install **VS Code** and **Git**.
- [ ] **Fork or clone** this repo and open it in VS Code.
- [ ] Make your **first commit**: tick this box, then run `git add . && git commit -m "start" && git push`.

## 1 · Build (web) — learn by making small things
Goal: feel how HTML, CSS, and JavaScript fit together.
- [ ] Read MDN's **"Getting started with the web"** (~2 hrs).
- [ ] Build a one-page **profile card** (your name, a photo, a few links) in plain HTML + CSS.
- [ ] Add one small bit of **JavaScript** — a dark-mode toggle, or a button that changes something.
- [ ] Start your **portfolio site v1** — even just a homepage. This is your real project.
- [ ] **Deploy it free** (GitHub Pages or Vercel) so it's live on the internet.

→ Full track: [`tracks/02-web`](tracks/02-web)

## 2 · Problem-solving — learn how to think
Goal: feel what "solving a problem" actually means.
- [ ] Solve **3 Easy** problems on LeetCode in **Python** (start: Two Sum, Valid Parentheses, Contains Duplicate).
- [ ] Re-solve **1** of them in **Java** — notice what stays the same (the logic) and what changes (the syntax).
- [ ] For each: try it yourself first, **AI off**. Stuck after 20–30 min? Read the *idea*, then re-code it yourself.

→ Full track: [`tracks/03-problem-solving`](tracks/03-problem-solving)
